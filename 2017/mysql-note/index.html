    Tips

    • 字串存在： LOCATE(你要查的字符串，对应的域) > 0
    • 字串连接：CONCAT(STR1, STR2, ...) ， CONCAT_WS 第一个参数是分隔符
    • 时间：FROM_UNIXTIME 和 UNIX_TIMESTAMP
    • 查看倒数第 20 到 30 笔数据：ORDER BY id DESC LIMIT 20, 30
    • 日期：CURDATE() = DATE(NOW())
